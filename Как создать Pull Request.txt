1. Создайте форк

Зайдите в мой репозиторий:
https://github.com/jaros3/cs02_homework
Вверху справа выпадающий список Forks. Нажмите create a new fork

2. Создайте пустой локальный репозиторий

git init

3. Настройте связку между локальным и удалённым репозиториями:

git remote add origin https://github.com/<ваш аккаунт>/<ваш форк>

4. Покомиттьте изменения

git add файлы
git commit -m "Я внёс такие-то изменения"

5. Отправьте локальные изменения в удалённый репозиторий

git push

6. Создайте Pull Request на сайте

Выберите ваш репозиторий. В верхнем меню найдите Pull Requests (две сливающиеся стрелки). Нажмите New Pull Request. Заполните тему и сообщение

7. Скопируйте ссылку из адресной строки браузера и отправьте на проверку.
